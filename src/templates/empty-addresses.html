Shared Address Books

Create a global shared address book for your organization. To share an address book with others, select the address book in the folder tree and choose Share Address Book from the folder pop-up menu. Enable Customize Sharing Properties option and click the Add button to add other users.

Your default private address book can be shared, but you can also create a separate one from the File > New Folder menu.
