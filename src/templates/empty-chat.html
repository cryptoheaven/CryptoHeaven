Secure Instant Messaging
Your secure instant message log is empty.

Send secure instant messages to anyone on your contact list. To send a new message, write it in the box below and click on the send icon. Your instant message may include file attachments.

Group Messaging

Any number of people can participate in an Instant Messaging session and they do not have to be online at the same time.

Add people to an existing chat session by selecting Invite to the Conversation from the pop-up menu.

Create a new multi-user chat session by highlighting all desired contacts and selecting Chat from the contacts pop-up menu.

Automatic Purging

Secure Instant Message log can be set to hold any number of messages. The default setting is to hold a maximum of 40 messages for up to 2 weeks. This can be changed by selecting Folder Properties > Purging from the folder pop-up menu.
