Groups and Role Based Access Control

CryptoHeaven Groups are designed to facilitate Role Based Access Control within your CryptoHeaven environment. Assign users to a group and set resource/folder access rights based on the group. Using groups to control access to shared resources simplifies administrative burden when adding or removing users. With group access controls simply assign a new user to a group and all previously set-up resources become available to the new member of the group.

Tips
A user can be a member of more then one group. A group can contain any number of members, including other groups.

Make sure to add everyone you work with to your Contact List. You need to have people in your Contact List in order to use advanced features such as secure instant messaging or secure file sharing.

Empty groups/roles are allowed. Create empty groups and use them to build your CryptoHeaven environment. You'll be able to add members later.
