CryptoHeaven keeps your unfinished messages in this folder.

The drafts message folder allows you to save all unfinished messages. If you are sending a message to someone and you don't have all the information available to finish the message then you can save it to drafts folder. You can then come back to it later at your convenience. To save an unfinished message you simply close the Compose Message window.
