The secure file folder is empty.

Take advantage of CryptoHeaven as your "digital safe deposit box." CryptoHeaven secure online storage capability allows you to access, store and backup digital information in a secure and private vault.

By default most CryptoHeaven file folders are private but you can also securely share them with people on your Contact List.

Create a new file folder by selecting File > New Folder.

Upload files with simple drag & drop or by clicking on theicon located on the toolbar.

Sharing File Folders

To share a folder right click on the folder and select Share Folder from the popup menu. Enable Customize Sharing Properties and Add people or groups to the list. Folders can be shared with anyone on your Contact List.

Only people whom you have explicitly granted permission to will be able to access files in a shared folder.
